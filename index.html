NanoTest

NanoTest is a extremely simple and super lightweight unit testing framework written in javascript.

Hopefully NanoTest will help those users who get intimidated by JsUnit and other testing frameworks to get used to TDD development :P

here's an example usage:

	var foo = 7;
	var bar = 4;

	function nanoTest_foo() {
		return NanoTest.assert('foo is not 7', foo == 7) || 0;
	};

	function nanoTest_bar() {
		return NanoTest.assert('bar is not 5', bar == 5) || 0;
	};
	
	var fails = NanoTest.runAll();
	for (fail in fails) {
		document.write("Failed: "+fails[fail]);
	}
